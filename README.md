# AMR-Tran
A small project for detecting AMR genes in draft assembled genomes of bacteria
# Pipeline
- Detecting AMR genes in draft assembled genomes by AMRFinderPlus (https://github.com/ncbi/amr)
- Writing 2 python scripts to process results from AMRFinderPlus
- Idenfifying the taxonomy of assembled genomes by GTDB-tk (https://github.com/Ecogenomics/GTDBTk)
- Finding variants in some genes that affect to AMR
